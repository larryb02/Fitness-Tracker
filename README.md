
# Install

First install dependencies:

npm v10.8.3  
node v22.9.0
react, reactDOM  
[golang v1.23.2](https://go.dev/dl/)  
[mySQL](https://dev.mysql.com/downloads/mysql/)

## Using MacOS 14.3

Use your preferred package manager to install npm:

`brew install node`

verify installation:

`npm`

`node`

Check to see if go is installed:

`go version`
